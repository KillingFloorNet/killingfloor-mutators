This mod gives Left 4 Dead-inspired skins and sounds to the specimens. This mod is fully client-side and works on-line! You're free to play in any of your favorite servers with perks enabled!

THIS MOD REQUIRES ADDITIONAL INSTALLATION STEPS. EITHER CHECK THE IMAGE ABOVE OR FOLLOW THE STEPS BELOW.

INSTALLATION INSTRUCTIONS
1. Subscribe to this item and run Killing Floor. Wait until you see the message that says this mod has been downloaded.

2. Close Killing Floor and then go to the following directory:
C:\Program Files (x86)\Steam\steamapps\common\killingfloor\

3. Copy the file "KF_EnemiesFinalSnd.uax" and paste it in the subsequent “Sounds” folder. Overwrite the original when asked.

4. Copy the file "KF_Specimens_Trip_T.utx" and paste it in the subsequent “Textures” folder. Overwrite the original when asked.

Congratulations, your specimens are now Left 4 Dead-ified!
