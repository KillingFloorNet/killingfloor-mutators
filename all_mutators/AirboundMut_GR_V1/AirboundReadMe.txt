Get Airbound, for all your air needs. Control Gravity, Jump Height, and Air Control; all within a single mutator.

Settings

-Use Default Gravity: Bypasses the custom gravity setting and will use the map's set gravity.

-Use Default Jump Height: Bypasses the custom jump height and uses the default of 230.

-Use Default Air Control: Bypasses the air control and uses the default of .15.

-Input Desired Gravity: Set gravity, range available from -2000 to 1.

-Input Desired Jump Height: Set height of jump, range avaiable from 0 to 1000

-Input Desired Air Control: Set how much control you have of your movement while in the air, range available from 0.0 to 5.0.
