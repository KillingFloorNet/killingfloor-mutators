This mutator is an advanced version of Faked Players. Allows to change fakes, zeds health, switch slomo on-off and many other things directly from the game.

Custom mode ('Chosen amount of fakes') allows a bit more than others. You can reserve player slots and use more than 5 fakes.

WhiteListed!

Console Commands
Commands are named pretty obvious so I'm listing them here for acquaintance, without many detailes.
If you want to find out what cmd does what - type 'mutate HELP' and 'mutate HELP <cmdname>' during a game. If you want to check what variables are currently used type 'mutate STATUS'.


Commands with ON-OFF switch:
LOCK
SPEC, SPECS, SPECTATOR, SPECTATORS
DRAMA, SLOMO
ADMINONLY, ADMIN
SOLO

Commands with Int values:
FAKED, FAKE, FAKES
PLAYER, PLAYERS, SLOT
HEALTH, HP
SPEC, SPECS, SPECTATOR, SPECTATORS
ReservedSlots, RS
ConfigFakes, CF

Commands that work as is:
HELP, HLP, HALP
STATUS
SKIP, SKP
SAVE
CREDITS
