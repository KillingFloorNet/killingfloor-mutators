Сейчас вот заморочился с WTF

Все ресурсы в один ukx пакет запихнул.
Удалил какие-то отсылки к SP и сделал просто пак с оружием и зомбиками
(То есть то, что я выложил в данном сообщении нельзя использовать как самостоятельный мутатор аналогичный SP)
Убрал зависимости пушек от перка. Мне при тесте оружия лень вспоминать какой должен быть перк, чтобы пушка полноценно работала.
Поэтому ссылка на перки осталась только в WTFEquipM79CF (для фаера поле огня, для остальных кластерный граник)
Каким-то пушкам я что-то по-мелочам добавил, поменял какие-то свойства, поправил косяки и т.д.
Например, банхаммеру добавил прыжки на правую кнопку мыши (фиг знает зачем)) )

Под ом описание оружия:

WTFEquipAFS12a
AA12 откидывающий, для подрыва - разрывные патроны. Спецскин

WTFEquipAK48S, WTFEquipBulldog, WTFEquipSCAR19a
Аналоги AK47, Bulldog, SCAR17 с доп. режимом (х3 скорость стрельбы)

WTFEquipBanHammer
Топор для подрыва. При ударе взрыв. Перезарядка по альтернативной атаке. Спецскин.
WTFEquipFireAxe
Топор для огника. Атака поджигает, альт атака поджигает со взрывом. На правой кнопке прыжок. Спецскин.

WTFEquipChainsaw
Бензопила с прыжком по правой кнопке мыши. Спецскин

WTFEquipKatana
Катана с замедлением времени по правой кнопке мыши. Работает во время волны

WTFEquipBoomStick
Двустволка в которой можно менять тип патрона. R - смена типа

WTFEquipShotgun
Дробовик в котором можно менять тип патрона. Во время перезарядки нажмите R

WTFEquipSawedOffShotgun
Двустволка. Мощная отдача. Спецскин

WTFEquipCrossbow
Фиг знает, не разбирался. Особой разницы не заметил. Какие-то изменения в стреле.

WTFEquipFT
Огник с альтернативной атакой. Широкий поток огня

WTFEquipFlaregun, WTFEquipGlowstick
Девайсы для подсветки

WTFEquipLethalInjection
Шприц замедляет и отравляет зомбиков. Кроме босса

WTFEquipM79CF
Граник M79 с полем огня для фаера. Для остальных кластерный взрыв

WTFEquipMP7M2a
Аналог MP7M, непрерывное лечение, скорость лечения изменена. И прочие мелочи

WTFEquipMachineDualies, WTFEquipMachinePistol
Пистолеты-пулемёты

WTFEquipNadeWeapon
В WTFEquipNadeFire можно поглядеть разные типы гранат. Не разбирался

WTFEquipPipeBomb
Практически обычная мина - разница в нюансах

WTFEquipRocketLauncher
WTF взрыв. Шумный и забавный

WTFEquipSA80a
SA80 винтовка

WTFEquipSelfDestruct
Мина для самоуничтожения. Спецскин

WTFEquipUM32a
По сути M32 c альтернативной атакой кидающей мины

WTFEquipWelda
Сварка. Можно сваркой наносить урон зомбикам
