Description:
This mutator fixes a severe bug related with demolition weapons.

Installation/Deinstallation:
Place KFExplosivesFix.u and KFExplosivesFix.ucl in your Killing Floor System folder.
To deinstall just remove those two files.

Permissions:
You are allowed to freely use Explosives Fix Mutator on any server you want and use the code for your own projects if you give credits.
You're not allowed to recompile and re-release this mod.
