Timestamp: 22:47 28/12/2009
Release: Beta 2
Version: 0.4
About: KF's deathmatch mayhem pits players against each other in the old fashioned free for all carnage. Only this time, specimens will lend a hand.
Author: PlateWolf <okushi@gmail.com>

Installation:
Drop all files on Killing Floor root directory (usualy Program Files\Steam\steamapps\common\killingfloor\).
Then add the ServerPackages line to the [Engine.GameEngine] section in your KillingFloor.ini (ServerPackages=KFDMMayhem).

This is a beta release, so please let me know any bugs encountered.
Hope you enjoy the mayhem !
